Overview 

Job Tracker App is a web application designed to help users manage and track jobs they are interested in applying to. Users can easily input job-related information and store personalized study notes for each position. 

Features 

Enter job title, company name, and job description. 

Add specific study notes related to each job. 

Intuitive web interface with simple text boxes for input. 

"Add To Job Tracker" button to save job entries quickly. 

Usage 

Launch the web application in your preferred browser. 

Fill out the text boxes with the desired job information: 

Job Title 

Company Name 

Job Description 

Study Notes (optional) 

Click the "Add To Job Tracker" button to save the entry. 

Repeat as needed to track additional jobs.
